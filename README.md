# TypeWriter

==简介==

TypeWriter是一款练习盲打兼学习英文的Windows软件。



==**OS**== 

Windows 7 and  above

==**.NET Framework**==  

.NET 8 and above

==**Thiry library**==

- Prism Library

- Xceed.WpfToolkit

- WpfColorFontDialog

- Hardcodet.NotifyIcon.Wpf



==背景==

编程6年发现自己还不会盲打，手速跟不上思路，严重影响了自己的编程水平。同时自己刚好在准备雅思考试，于是便基于WPF编写了这款，可加载英文素材文本练习打字的软件。
